# Usage

## Service

To start the service:

```bash
cd service
docker-compose up --build -d
```

To stop the service:

```bash
docker-compose down
```

## Checker

To start the checker:

```bash
cd checker
docker-compose up --build -d
```

To stop the checker:

```bash
docker-compose down
```
